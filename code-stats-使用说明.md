# Code Stats Skill 使用说明

## 简介

这是一个用于快速统计项目代码和文档行数的 Claude Code skill。

## 安装

将 `code-stats.skill` 文件安装到 Claude Code：

```bash
claude code skill install code-stats.skill
```

## 使用方法

安装后，你可以直接向 Claude 提出统计请求：

- "统计这个项目的代码行数"
- "帮我看看代码和文档有多少行"
- "分析项目代码量"
- "快速估算代码规模"

Claude 会自动识别并使用这个 skill 来完成统计。

## 功能特性

### 文件分类

**代码文件**：
- 编程语言：Python, Java, JavaScript, TypeScript, C/C++, Go, Rust 等
- 配置文件：JSON, YAML, XML, TOML 等
- 样式文件：CSS, SCSS, SASS, LESS
- 标记语言：HTML, XML

**文档文件**：
- Markdown (.md)
- 文本文件 (.txt, .rst, .adoc)
- 办公文档 (.pdf, .doc, .docx)

### 统计规则

- **包含所有行**：代码、注释、空行全部统计
- **快速估算**：适合快速了解项目规模，不追求完全精确
- **自动排除**：自动排除 node_modules、.git、dist、build 等常见目录

### 输出内容

1. 代码文件统计（按文件类型分类）
2. 文档文件统计（按文件类型分类）
3. 其他文件统计（可选）
4. 总计汇总

## 高级用法

如果需要手动运行脚本（不通过 Claude）：

```bash
# 统计当前目录
python scripts/count_lines.py

# 统计指定目录
python scripts/count_lines.py /path/to/project

# 排除额外的目录
python scripts/count_lines.py --exclude temp cache

# 不使用默认排除列表
python scripts/count_lines.py --no-default-exclude
```

## 注意事项

- 统计包括所有行（空行、注释、代码）
- 默认排除常见的依赖和构建目录
- 对于大型项目，扫描可能需要几秒钟
- 二进制文件会被跳过
- 使用 UTF-8 编码读取文件
