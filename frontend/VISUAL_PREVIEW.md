## 🎨 设计理念

**风格定位**: 现代商务风格 - 专业、简洁、科技感

**色彩体系**:
- 主色调: 深蓝紫渐变 (#667eea → #764ba2)
- 辅助色: 粉红、天蓝、翠绿渐变

**设计语言**: 扁平化 + 微妙阴影 + 流畅动画

## 🎭 动画效果

### 页面加载动画
```
登录页:
  背景: 渐变循环 (15s)
  左侧: 从左滑入 (0.8s)
  右侧: 从右滑入 (0.8s)
  特性: 逐个淡入 (0.1s 延迟)

主页:
  导航栏: 从上滑下 (0.5s)
  内容: 淡入上浮 (0.6s)
  卡片: 逐个出现 (0.1s 延迟)

会议室:
  顶栏: 从上滑下 (0.5s)
  视频: 淡入 (0.6s)
  控制栏: 从下滑上 (0.5s)
```

### 交互动画
```
悬停效果:
  - 卡片上浮 (-8px)
  - 阴影增强
  - 图标旋转/放大
  - 颜色渐变

点击效果:
  - 按钮下压
  - 涟漪扩散
  - 状态切换

状态指示:
  - 录制红点脉冲
  - 说话边框闪烁
  - 加载旋转
```

## 📐 布局系统

### 间距规范
```css
xs: 4px   - 小间距（图标间距）
sm: 8px   - 标准间距（元素内边距）
md: 16px  - 中等间距（卡片内边距）
lg: 24px  - 大间距（区块间距）
xl: 32px  - 超大间距（页面边距）
```

### 圆角规范
```css
sm: 4px   - 小元素（标签）
md: 8px   - 标准元素（输入框）
lg: 12px  - 大元素（按钮）
xl: 16px  - 卡片（容器）
2xl: 20px - 大卡片
full: 50% - 圆形按钮
```

### 阴影层次
```css
sm: 0 2px 8px rgba(0,0,0,0.06)   - 轻微阴影
md: 0 4px 20px rgba(0,0,0,0.1)   - 标准阴影
lg: 0 8px 40px rgba(0,0,0,0.15)  - 明显阴影
xl: 0 12px 60px rgba(0,0,0,0.2)  - 强烈阴影
```
