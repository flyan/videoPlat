## 技术实现细节

### 设计系统
```css
/* 主色调 */
primary: #667eea (紫蓝色)
success: #4CAF50 (绿色)
warning: #FF9800 (橙色)
danger: #EF4444 (红色)

/* 渐变色 */
purple: #667eea → #764ba2
pink: #f093fb → #f5576c
blue: #4facfe → #00f2fe
green: #43e97b → #38f9d7

/* 间距系统（8px 基准）*/
xs: 4px, sm: 8px, md: 16px, lg: 24px, xl: 32px

/* 圆角系统 */
sm: 4px, md: 8px, lg: 12px, xl: 16px, full: 9999px
```

### 动画效果
- **淡入动画**: 0.6s ease-out
- **滑入动画**: 0.5s ease-out
- **缩放动画**: 0.5s ease-out
- **悬停动画**: 0.3s ease
- **脉冲动画**: 1.5-2s infinite

### 响应式断点
- 桌面端: > 1024px
- 平板端: 768px - 1024px
- 移动端: < 768px
- 小屏移动端: < 375px
