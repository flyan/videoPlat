### 后端编译需要
- **Java 17+**（当前系统是 Java 8）
- **Maven 3.6+** 或使用 Maven Wrapper

### 解决方案

#### 方案 1: 安装 Java 17（推荐）
1. 下载 Java 17: https://adoptium.net/
2. 安装并配置环境变量
3. 验证: `java -version`

#### 方案 2: 使用 IDE 编译
1. 使用 IntelliJ IDEA 打开 `backend` 目录
2. IDE 会自动下载依赖并编译
3. 直接在 IDE 中运行 `VideoplatApplication.java`

#### 方案 3: 使用 Docker
