## 设计特点

### 配色方案（活力创新风格）
- **主色调**: #6366F1（活力紫）- 用于标题、主要元素

- **标题**: 40-66pt，粗体

## 修改和定制

### 如何修改 PPT

1. **修改内容**
   - 直接在 PowerPoint 中编辑文字
   - 保持原有的配色和布局风格

2. **修改配色**
   - 如需更换配色方案，建议统一修改所有页面
   - 保持配色的一致性

3. **添加页面**
   - 可以在现有页面之间插入新页面
   - 建议复制现有页面作为模板

4. **重新生成**
   - 如需大幅修改，可以编辑 `generate-ppt.js` 脚本
   - 运行 `node generate-ppt.js` 重新生成

### 生成脚本说明

**文件**: `generate-ppt.js`

**依赖**:
```bash
npm install pptxgenjs
```

**运行**:
```bash
node generate-ppt.js
```

**修改配色**:
在脚本开头修改 `COLORS` 对象：
```javascript
const COLORS = {
  primary: '6366F1',      // 主色调
  secondary: '14B8A6',    // 辅助色
  accent: 'F59E0B',       // 强调色
  success: '10B981',      // 成功色
  background: 'FFFFFF',   // 背景色
  lightBg: 'F8FAFC',      // 浅灰背景
  text: '1E293B',         // 文字颜色
  textLight: '64748B'     // 浅色文字
};
```

## 导出和分享

### 导出为 PDF
1. 在 PowerPoint 中打开 PPT
2. 文件 → 另存为 → 选择 PDF 格式
3. 保存为 `VideoPlat-项目介绍.pdf`

### 导出为图片
1. 文件 → 导出 → 更改文件类型
2. 选择 PNG 或 JPEG 格式
3. 每页将保存为单独的图片文件

### 在线分享
- 上传到 SlideShare
- 上传到 Google Slides
- 转换为视频格式（使用 PowerPoint 的"导出为视频"功能）

## 注意事项

1. **字体兼容性**
   - PPT 使用 Arial 和 Arial Black 字体
   - 在其他设备上打开时，如果缺少字体，系统会自动替换

2. **图标显示**
   - PPT 中使用了 emoji 图标
   - 在某些旧版本软件中可能显示为方框
   - 建议使用 PowerPoint 2016 及以上版本

3. **文件大小**
   - 当前文件大小约 214 KB
   - 如果添加图片，文件大小会增加

4. **版本控制**
   - 建议保留生成脚本 `generate-ppt.js`
   - 便于后续修改和重新生成
