# VideoPlat 项目介绍

## 📊 项目概况

**项目名称**: VideoPlat - 在线视频会议平台
**项目定位**: 类似 Zoom 的视频会议系统
**当前状态**: Beta 测试版本
**开发阶段**: 核心功能已完成，部分高级功能待开发

---

## 🎯 项目是做什么的？

VideoPlat 是一个**在线视频会议系统**，让用户可以：
- 创建和加入视频会议室
- 进行多人实时视频通话（最多10人）
- 录制会议内容
- 回放历史会议录像
- 实时文字聊天
- 屏幕共享

**使用场景**：远程办公、在线教育、视频面试、团队协作等

## 🏗️ 框架架构

### 整体架构
```
┌─────────────┐      HTTPS/WSS      ┌─────────────┐
│   浏览器     │ ◄─────────────────► │   Nginx     │
│  (React)    │                      │  (反向代理)  │

                                            │
                    ┌───────────────────────┼───────────────────────┐
                    │                       │                       │
            ┌───────▼────────┐      ┌──────▼──────┐       ┌───────▼────────┐
            │  Spring Boot   │      │   Redis     │       │  PostgreSQL    │
            │   (后端 API)    │◄────►│   (缓存)     │       │   (数据库)      │
            └────────────────┘      └─────────────┘       └────────────────┘
                    │

            │  Agora 声网     │
```

### 前端架构（React）
- **UI 框架**: React 18 + Ant Design
- **状态管理**: Zustand（轻量级状态管理）
- **路由**: React Router
- **样式**: Tailwind CSS + CSS Modules
- **构建工具**: Vite（快速开发和构建）
- **WebRTC**: Agora Web SDK（音视频通信）
- **视频播放**: Video.js（录制回放）

### 后端架构（Spring Boot）
- **核心框架**: Spring Boot 3.x
- **安全认证**: Spring Security + JWT Token
- **数据访问**: Spring Data JPA（ORM）
- **API 文档**: SpringDoc OpenAPI（Swagger）
- **实时通信**: Spring WebSocket（信令和聊天）
- **数据库**: PostgreSQL（Neon 云数据库）
- **缓存**: Redis（会话管理）

### 部署架构
- **容器化**: Docker + Docker Compose
- **Web 服务器**: Nginx（反向代理 + HTTPS）
- **前端**: 静态文件部署
- **后端**: Spring Boot 独立运行
- **数据库**: Neon PostgreSQL（云端）
- **缓存**: Redis（本地或云端）

## 🚀 快速启动

### 使用 Docker（推荐）
```bash
# 1. 配置环境变量
cp .env.example .env

# 2. 启动所有服务
docker-compose up -d

# 3. 访问应用
# HTTPS: https://192.168.10.9 或 https://localhost
```

### 本地开发
```bash
# 前端
cd frontend
npm install
npm run dev  # http://localhost:5173

# 后端
cd backend
mvn spring-boot:run  # http://localhost:8080
```
